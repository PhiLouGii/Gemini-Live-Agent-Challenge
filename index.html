Install Grandma Mode

Free to use. Works on Chrome and Microsoft Edge. No account needed.

1

Download the extension

Click the button below to download the Grandma Mode zip file.

2

Unzip the file

Right-click the downloaded file and select Extract All to unzip it.

3

Open Extensions page

In Chrome or Edge, go to chrome://extensions or edge://extensions

4

Enable Developer Mode

Toggle on Developer Mode in the top right corner of the extensions page.

5

Load the extension

Click Load Unpacked and select the unzipped grandma-mode folder.

6

You're done!

Click the Grandma Mode icon in your toolbar to open the side panel and start browsing smarter.
